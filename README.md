## What This Is

A structured, self-paced curriculum from **using AI** to **modifying and deploying AI** —
built specifically for developers who already know how software works and want to stay ahead
of a fast-moving field.

This is not a survey course. Every lesson produces something you can run, show, or explain
in an interview. Every phase builds on the one before it.

## How to Use This Book

Each lesson is a self-contained HTML file — open it in a browser, read it, run the code.
The `Learn/` folder is where your practice code lives. After each lesson, write your version there.

**Lesson** → read and understand  
**Learn/** → write and run  
**Interview Spotlight** → know cold
