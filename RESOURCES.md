# Resources

## Phase 1 — AI Builder (Using Models Expertly)

| Resource | Type | Why |
|----------|------|-----|
| [Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook) | GitHub / Code | Best practical patterns for agents, tool use, RAG with Claude |
| [LangChain Docs — Agents](https://python.langchain.com/docs/modules/agents/) | Docs | Reference implementation of the agent loop |
| [Building LLM-Powered Applications (Chip Huyen)](https://huyenchip.com/2023/05/02/rag.html) | Article | RAG architecture decisions, production-grade |
| [Pinecone Learn — Vector DBs](https://www.pinecone.io/learn/vector-database/) | Guide | Best intro to vector databases for developers |
| [OpenAI Function Calling Guide](https://platform.openai.com/docs/guides/function-calling) | Docs | Canonical reference for tool/function calling patterns |
| [Weaviate Academy](https://weaviate.io/developers/academy) | Course | Free, hands-on RAG from a vector DB company |

## Phase 2 — How Models Work

| Resource | Type | Why |
|----------|------|-----|
| [Attention is All You Need (original paper)](https://arxiv.org/abs/1706.03762) | Paper | The transformer architecture — required reading |
| [The Illustrated Transformer (Jay Alammar)](https://jalammar.github.io/illustrated-transformer/) | Blog | Best visual explanation of attention — start here |
| [Andrej Karpathy — Neural Nets Zero to Hero](https://karpathy.ai/zero-to-hero.html) | YouTube | Builds a GPT from scratch in Python — best practical course |
| [fast.ai Practical Deep Learning](https://course.fast.ai/) | Course | Code-first approach to DL — aligns with Abhinav's preference |
| [The Little Book of Deep Learning (François Fleuret)](https://fleuret.org/public/lbdl.pdf) | Book | Short, rigorous, free — fills gaps fast |
