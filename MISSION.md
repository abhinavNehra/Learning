# Mission: AI Engineering — From User to Expert

## Why
Abhinav is a senior full-stack developer (11 yrs, React/Node.js/Python) who has completed a prompt engineering foundation and wants to become a deeply capable AI engineer. The goal is not just to use AI APIs — it's to understand, modify, deploy, and lead AI systems at a production level. This unlocks the ability to ship AI products, get hired as an AI/ML engineer, or own AI strategy at a company.

## Success looks like
- Build and ship a production AI application (agents, RAG, tool use) — not a toy
- Explain how a transformer works to a non-engineer, and to a senior ML engineer
- Fine-tune an open-source model (LLaMA / Mistral) on a custom dataset
- Serve a model with vLLM or similar and benchmark its throughput
- Run an end-to-end eval pipeline on a prompt change before shipping
- Stay current: read and extract value from new AI research papers within a week of release

## Constraints
- 4–8 hrs/week (serious pace — evenings + weekend blocks)
- Full-time job alongside learning — lessons must be completable in one sitting
- Prefers hands-on code first, theory second
- React/Node.js/Python stack — examples should leverage existing expertise
- Prompt engineering curriculum (6 modules) already complete — do not re-teach

## Out of scope
- Training models from scratch on raw data (research territory, not engineering)
- Pure ML theory without implementation (save for Phase 2 — only what's needed)
- Chasing framework churn — teach concepts that survive framework changes
